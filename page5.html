Iframes

An Iframe allows an HTML document to display content from other HTML documents. Iframes are produced by the <iframe> element and the height and width of iframes can be edited.

The Iframe Syntax

  • <iframe src="url" title="description"></iframe>

Height & Width Adjusting

  • <iframe src="url" title="description" height="[#]" width="[%]"></iframe>
