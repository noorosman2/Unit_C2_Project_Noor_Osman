What is HTML?

HTML stands for HyperText Markup Language. HTML is the code language that makes the World Wide Web possible and it was invented by Tim Berners-Lee in 1989. HTML markups are called tags which are shown by angle brackets (< and >). HTML can be written using NotePad on Windows, TextEdit on Mac, as well as websites such as Codepen.io.
