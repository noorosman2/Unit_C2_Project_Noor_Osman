Headings, Paragraphs, Tables, & Head Elements

Headings

Headings allow for structure to be added to a webpage. They span from <h1> to <h6> and get smaller to show subheadings.

Paragraphs

Paragraph <p> is where the majority of text on a webpage goes. <p> is used for formatting and to include line breaks within a paragraph, the tag <br> can be used. The <pre> tag can also be used when writing preformatted text.

Tables

Tables are the best way for web developers to organize data and the tags are <table> to create a table, <tr> to create a table row, <th> to create a table heading cell, and <td> to insert table data. Table borders can also be changed and their boldness, thickness, and color can all be edited. In order to have a cell span multiple rows or columns, the tags <th colspan="#"> and/or <td rowspan="#"> can be used.

HTML Structure of a Table

<table>
      <tr>
        <th>Heading 1</th>
        <th>Heading 2</th>
      </tr>
      <tr>
        <td>Data 1</td>
        <td>Data 2</td>
      </tr>
    </table>

Example of a Table

Data Data Data
Data Data Data
