Entities

An entity is a way to express characters on a document that would otherwise be turned into a part of the code.

Character Encoding Schemes

  • American Standard Code for Information Interchange (ASCII): The first character encoding scheme and the one that was most widely used before Unicode.
  • Extended Binary Coded Decimal Interchange Code (EBCDIC): Before IBM converted to ASCII in the 1980s, EBCDIC was used by mainframe computers.
  • American National Standards Institute (ANSI) or Windows-1252: ANSI is an extended version of ASCII that was developed by ANSI and used by Windows until Unicode.
  • ISO-8859: The International Organization for Standardization developed ISO-8859 and it was different formats that existed before Unicode.
