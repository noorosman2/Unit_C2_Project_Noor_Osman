Comments, Attributes, & Links

Although comments are hidden in an HTML document, they’re still very helpful as they can be used to place reminders or instructions that can’t be on the document itself. Attributes appear within a tag and allow for the text’s appearance to be modified. This includes size and color.

Comments

To leave a comment, type "<!--" before the comment and ">--!" after it. Elements are also disabled when left in comments.

Attributes

Attributes are codes within tags that modify the tags. This can be size or color. The standard syntax for an attribute is: <tagName attributeName"value">content</tagName>.

Links

The anchor, <a>, is the element used to make a hyperlink to another location. The <href> attribute is necessary to locate the link. The general syntax for a link is <a href="pathToFile">LinkText</a>.

Examples

Absolute/External Link

<a href="[Website Link]">[Website Name]</a>

Relative/Internal Link

<a href="[File Link]">[File Name]</a>

More Link Attributes

Download

Lets user download a file <a download="[File Name]">

Target

Specifies where in the page the link will open <a href="[URL]" target="[Target]">

Some Targets:
  • _blank: opens in a new window/tab
  • _top: opens at the top of a page
  • _self: opens the same iframe
  • _parent: opens within the parent of the frame

Images

An image can be embedded into a webpage using the <img> element. The src attribute has to be used to locate the image and the alt attribute is used to describe the image for screen readers.

Sizing Images

The width and height of an image can be edited and are specified in pixels. <img src="file" width="200">

Local Image

<img src"[Image Link]" alt="[Image Title]">

Remote Image

<img src="[Web Link]" alt="[Image Title]">
