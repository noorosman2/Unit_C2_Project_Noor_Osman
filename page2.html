Formatting Tags & Lists

What Are Formatting Tags?

Formatting tags are tags that change the appearance of text and can be used to modify from a character to a line of text. List elements allow for different lists to be added to text, these include both unnumbered and numbered lists.

Formatting Tags

  • Bold <b>
  • Bold <strong>
  • Italics <i>
  • Italics <em>
  • Highlight <mark>
  • Smaller Text <small>
  • Crossed Out Text <del>
  • Underline <ins>
  • Subscript <sub>
  • Superscript <sup>

List Elements

For each line of the list, it's necessary to re-enter the attribute.

  • Ordered/Numbered List <ol>
  • Unordered/Bulleted List <ul>
  • Both use <lt>, list item, for the items within the lists.
  • <ol>, <ul>, and <li> are all block-level elements.

Example

<ol>
        <li>list</li>
        <li>is</li>
        <li>numbered</li>
      </ol>
      
      <ul>
        <li>list</li>
        <li>has</li>
        <li>bullets</li>
      </ul>
